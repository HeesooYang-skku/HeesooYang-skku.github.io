Final Project

A dashboard that analyzes contemporary art exhibition titles as artistic and institutional language.

Exhibition Title Language Dashboard

This Streamlit web app analyzes Korean contemporary art exhibition titles by collecting data such as title, venue, year, language type, main keywords, category, and source URL.

The goal is to reveal patterns in how exhibitions are introduced: which words appear often, whether institutions use Korean, English, or mixed-language titles, and how keywords differ across venues.

Python Pandas Streamlit Plotly / Matplotlib GitHub

04

Compares keyword patterns across museums and galleries, helping users explore institutional differences.

Development Plan

The project is developed in three steps, from basic layout to interactive charts and polished real data.

Step 1

Basic Layout + Sample Data

Create the Streamlit app structure, add a project introduction, build a sample exhibition dataset, and display it as a table.

Step 2

Charts + Interactivity

Add word frequency, language type, and venue comparison charts. Use sidebar filters for year, venue, and language type.

Step 3

Polish + Real Data

Collect real exhibition data, refine keyword categories, improve the visual design, add source URLs, and test the final dashboard.
